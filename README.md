This is a simple _boilerplate project layout_ for **running custom web process on [ep.io][]**

[ep.io]: http://www.ep.io
---

- [epio.ini]

>platform specified configuration file

- requirements.txt

>[pip requirements file]

- app.py

>WSGI app built on [Flask web framework]

- server.py

>[HTTP server] using [tornado]


[epio.ini]: https://www.ep.io/docs/epioini/ "epio.ini options"

[pip requirements file]: http://www.pip-installer.org/en/latest/requirement-format.html
"The requirements file format"

[Flask web framework]: http://flask.pocoo.org/
"Flask is a microframework for Python based on Werkzeug and Jinja 2"

[tornado]: http://www.tornadoweb.org/ "Tornado Web Server"

[HTTP server]: http://www.tornadoweb.org/documentation/httpserver.html
"tornado.httpserver ¡X Non-blocking HTTP server"
